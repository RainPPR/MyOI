# 对拍

## 数据生成器：gen

```cpp
// generate：生成器
#include <cstdio>
#include <cstdlib>
#include <ctime>

using namespace std;

// [l, r] random number
int rnd(int l, int r)
{
	return (long long) rand() * rand() % (r - l + 1) + l;
}

int main()
{
	srand(time(0));
	freopen("ab.in", "w", stdout);
	int a = rnd(1, 150), b = rnd(1, 150);
	printf("%d %d", a, b);
	return 0;
}
```

## 文件比对：fc

```shell
fc ab1.out ab2.out /n
```

## 原理

`gen.exe` 产生数据：写入文件
产生标准答案：执行暴力算法
产生其他答案：执行其他算法
比对输出的文件：错误则中断

## 对拍自动化：系统命令

```cpp
// 对拍程序
#include <windows.h>

using namespace std;

int main()
{
	while (1)
	{
		system("gen.exe");
		system("baoli.exe");	// ab.ans
		system("ab.exe");		// ab.out

		system("fc ab.ans ab.out /n");
		if (system("fc ab.ans ab.out"))
			system("pause");
		
		Sleep(500);
	}
}
```
