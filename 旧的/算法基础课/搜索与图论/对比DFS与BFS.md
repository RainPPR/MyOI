# 对比DFS与BFS
| 算法  | 实现方式 | 空间复杂度 |                        特性、应用                        |
| :--: | :------: | :-------: | :--------------------------------------------------: |
| DFS  |    栈    | $O(n)$	  | 不具备最短路性质；搜索树；回溯--完全恢复原状、顺序--字典序、剪枝--提前判断分支不合法 |
| BFS  |   队列   | $O(2^n)$  |                       最短路、最优解                       |
