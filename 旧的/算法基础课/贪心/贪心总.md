# 贪心
先尝试、猜做法，再证明
选择当前最好的情况走，如果贪心正确，就可以走到最优解。
## 证明两个值相等的策略：
需证明 $a=b$：
先证明 $a \ge b$，再证明 $a \le b$
## 贪心策略集合
1. $sum = a_1t_1+a_2t_2+a_3t_3+\dots+a_nt_n\text{ }\text{ }(a_i>a_{i+1})$
策略：$t$ 数组从小到大排序，$sum$ 最小；$t$ 数组从大到小排序，$sum$ 最大。
2. $m = |a-x|+|b-x|\text{ }\text{ }(a \le b)$，使 $m$ 最小
策略：$a \le x \le b$
