# 记忆化搜索
用记忆化搜索可以实现DP，思路更清晰！
```cpp
int f[...];		// 实现DP的数组，几个维度看题目分析
int dp(...)		// 记忆化搜索函数，参数看题目分析
{
	int &v = f[...];	// 传到函数的参数对应的f数组
	if (v != -1)		// 如果已经搜索过，直接返回已计算的结果
		return v;
	v = ...;			// 根据属性与题意给v提前赋一个基础值
	for (...)			// 枚举这一情况的所有可能性
		if (...)		// 这一分支满足条件
			v = ...;	// 计算值：
			// 最大值用：v = max(v, ..[dp(...)]..);
			// 最小值用：v = min(v, ..[dp(...)]..);
			// 计数量用：++v;
			// 具体情况具体分析
	return v;
}
```
